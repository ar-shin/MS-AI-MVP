# MVP - AI 기반 RFP 분석 Agent

## 필요 사항
- Azure Document Intelligence(Recognizer)
- Azure OpenAI API
- Azure AI Search
- Python 3.9+
- Streamlit

## 실행 방법
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
3. `가상환경은 venv\Scripts\activate 후 run`
